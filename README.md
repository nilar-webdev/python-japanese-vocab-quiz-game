# Python Japanese Vocabulary Quiz Game

This is a simple command-line quiz game for learning Japanese vocabulary.

## Features

* Shows Japanese vocabulary questions
* User types the English meaning
* Randomly selects quiz questions from `vocab.json`
* Checks correct and wrong answers
* Shows the final score
* Allows the user to play again
* Saves the highest score locally

## Technologies Used

* Python
* JSON

## How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python main.py
```

If `python main.py` does not work, try:

```bash
py main.py
```

## Purpose

The purpose of this project is to practice basic Python programming skills, including variables, functions, loops, conditionals, lists, dictionaries, JSON file handling, and simple game logic.

This project also helps me practice Japanese vocabulary while improving my Python skills.
